Supervised vs Unsupervised Machine Learning

Samuel D.
Follow
Mar 26, 2020 ¬∑ 5 min read

Supervised and Unsupervised learning are two major key machine learning concepts which are used in solving, or performing specific tasks, by learning from experiences and/or the relationships between the data. But both techniques are used in different scenarios and with different datasets. The main difference between supervised and unsupervised learning is the fact that supervised learning involves training prelabeled inputs to predict the predetermined outputs. Meaning the goal of supervised learning is to learn a function that, given a sample of structured data the model is trained to produce the desired results. Supervised learning best approximates the relationship between the input and output, observed in the data. And on the contrary unsupervised learning deals with unlabeled data, input and output, instead it‚Äôs goal is to discovers patterns and relationships between the data.

Supervised Learning

Supervised machine learning needs supervision to train the model, hence the name. A great analogy I came across is a student learns new materials in the presence (or supervision) of a teacher. With supervised learning, the goal is to predict the outputs of new data on the basis of the trained prelabeled data. Supervised learning can be used for two types of problems, Classification (classifying data into predetermined groups) and Regression (predicting values from numerical data).

Classification

Classification algorithms are used when the output variable is categorical, which means data is grouped into two or multiple classes such as Yes-No, True-false, Children-Adults-Elderly, Circles-Squares-Triangles, etc. In the real-world, supervised learning can be used for Fraud Detection, Image Classification, Risk Assessment, Spam Filtering, etc.

Below are some popular Classification Models:

  • Random Forest
  • Decision Trees
  • Logistic Regression
  • Support Vector Machines
  • Convolution Neural Networks (image deep learning)

Regression

Regression algorithms are used if there is a relationship between the input variable and the output variable. It is used for the prediction of continuous (numerical) variables, such as Weather forecasting, Market Trends, etc.

Below are some popular Regression algorithms which come under supervised learning:

  • (Simple and Multiple) Linear Regression
  • Regression Decision Trees
  • Ridge or Lasso Linear Regression
  • Bayesian Linear Regression
  • Polynomial Regression

There are some disadvantages to using supervised learning. Data needs to be cleaned and labeled, data found in the real world is rarely labeled. Training required lots more computational power/time; as opposed to unsupervised learning. Supervised learning models are not suitable for handling overly complex tasks. Supervised learning can poorly predict the output if the test data is severely different from the training set.

Unsupervised Learning

Unsupervised machine learning does not need any supervision (there is no teacher to this); hence the name again. With unsupervised machine learning the goal is not to find the ‚Äúmost correct‚Äù answers; as the model will not be trained to know what is considered correct or incorrect. Instead its goal is to infer patterns/relationships from unlabeled inputted data. It finds patterns from the data by its own, instead of from labeled data. With supervised learning, the model can predict the outputs on the basis of prior experiences. A great analogy I came up with is there‚Äôs multiple routes home from school, a child can take, and after many different trials and errors the kid finds out the fast routes are made by cutting through alleyways and side streets. Unsupervised learning can be used for those cases where we have only input data and no predetermined corresponding output data. Unsupervised learning can be used for two types of problems, Clustering (groups data based on similarities) and Association (discovering the exact rules that will describe the patterns of your data).

Clustering

Clustering is a method of grouping the objects into clusters such that objects with most similarities remains into a group and has less or no similarities with the objects of another group, and are displayed farther away. Cluster analysis finds the commonalities between the data objects and categorizes them as per the presence and absence of those commonalities.

Below are some popular Clustering algorithms which come under unsupervised learning:

  • K-Means Clustering
  • Mean-Shift Clustering
  • Hierarchical Clustering
  • Expectation‚ÄìMaximization (EM) Clustering using Gaussian Mixture Models (GMM) or EM GMM
  • Deep Neural Networks

Association

An association rule is an unsupervised learning method that is used for finding the relationships between variables in the large database. It determines the set of items that occurs together in the dataset. Association rule makes marketing strategy more effective. Such as people who buy X item (suppose a bread) are also tend to purchase Y (Butter/Jam) item. A typical example of Association rule is Market Basket Analysis.

Below are some popular Association algorithms which come under unsupervised learning:

  • Learning Classifier System
  • Association Rule Learning
  • Apriori Algorithm
  • Singular-Value Decomposition

There are some disadvantages to using unsupervised learning. It also doesn‚Äôt work that well with overly narrowly tasks or with small datasets. Unsupervised learning models may give less accurate result as compared to supervised learning, due to do not knowing the exact output in advance. With no clear cut answer, the outputs are very open to interpretation.
